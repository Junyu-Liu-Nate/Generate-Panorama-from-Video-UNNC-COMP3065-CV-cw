## Notes

- Most execution commands/explainations are included in ```excecute_basic.ipynb```.
- All running results are precerved on the code blocks in ```excecute_basic.ipynb```.
- When running the code, users may be prompt to enter customized settings. Please follow the prompt messages.
- When a panorama stitching program finishes, the generated panorama will be located in img/:
  - The '_final.jpg' is the raw output of the panorama.
  - The '_crop.jpg' is the loose crop version (preseve all contents) of the panorama, generated by automatically locating and croping. 
  - The '_tightcrop.jpg' is the tight crop version (as little black space as possible) of the panorama, generated base on loose crop version.
  - If use set ```if_write_frame=True``` in code prompts, there will be subfolders contains results for each frame.
- Due to file size limitations of the uploaded zip file, the trained NAFNet-based denoising model is not included in this folder (it has a size of 271.8 MB). There is a download link in ```excecute_basic.ipynb```, and users should follow the instructions to download it and place it at correct location if they want to run the code.

## Environments and packages
The environment for executing ```Generate panorama (No deblurring version)``` and ```Generate panorama (Deconvolution-based method)``` code blocks in ```excecute_basic.ipynb``` is given in ```environment.yml```. The recommended excecution environment for ```Generate panorama (NAFNet-based method)``` code blocks in ```excecute_basic.ipynb``` is Google Colab.

Use ```conda env create -f environment.yml``` to create the environment.

If there are errors when all the above packages are installed, please refer to the Appendix for more information to check for details.
